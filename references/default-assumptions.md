# Default Assumptions for Preliminary Kazakhstan HPP Screening

Use only when project-specific data is missing. Label all such values as assumptions.

| Parameter | Default Range |
|---|---:|
| Construction period | 4-6 years for 50-200 MW reservoir HPP |
| Operating life | 30-40 years for financial model, longer for economic life |
| CAPEX intensity | USD 2.2-3.5 million/MW for medium/high-head reservoir HPP |
| OPEX | 1.5%-2.5% of CAPEX per year |
| Civil contingency | 10%-20%, higher if geology immature |
| P75 generation | 90%-94% of P50 |
| P90 generation | 80%-88% of P50 |
| Availability | 92%-97% after commissioning |
| Debt ratio | 60%-75% if bankable offtake exists |
| USD debt interest | 7%-10% depending on lender mix |
| Debt tenor | 12-18 years |
| Equity target IRR | 13%-18% USD for emerging-market greenfield HPP |
| Discount rate | 9%-12% real/nominal depending on currency basis |
| FX sensitivity | Test KZT depreciation of 10%-30% |
| Construction delay | Test 6, 12, and 24 months |

For capacity-market reverse engineering, calculate required KZT/MW-month capacity price under multiple energy tariff assumptions. Treat energy revenues as upside unless the settlement and curtailment rules are clear.
